# ProCards Feedback

This repository is the public feedback hub for ProCards.

Use it to:

- report bugs
- request features
- browse existing product feedback

## Quick Links

- [Report a bug](https://github.com/ProCards-AI/procards-feedback/issues/new?template=bug_report.yml)
- [Request a feature](https://github.com/ProCards-AI/procards-feedback/issues/new?template=feature_request.yml)
- [View open issues](https://github.com/ProCards-AI/procards-feedback/issues)
- [Open the feedback portal](https://procards-ai.github.io/procards-feedback/)

## Before You Post

- Search existing issues first to avoid duplicates.
- Include clear steps, screenshots, and device details for bug reports.
- Keep in mind that issues in this repo are public.

## Need Help Instead?

If you need account help, billing help, or direct support, email:

- `support@procards.ai`

## What Belongs Here

### Good fits

- reproducible bugs
- product feature requests
- workflow pain points
- UI or usability feedback

### Better for support email

- account-specific problems
- billing questions
- anything with private personal information
